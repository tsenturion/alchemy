# Алхимия Skyrim

Русский справочник по алхимии Skyrim: ингредиенты, эффекты, зелья, яды и совместимые сочетания.

Открыть сайт: <https://tsenturion.github.io/alchemy/>

## Как пользоваться

- Включайте нужные дополнения в меню "Дополнения".
- Ищите ингредиенты по названию через поле поиска.
- Нажимайте "Выбрать Эффект", чтобы посмотреть все ингредиенты с нужным эффектом.
- Можно также нажать на эффект прямо в таблице.
- Нажмите на название ингредиента, чтобы добавить его в выбранные.
- Красная часть используется для варки ядов, зеленая - для варки зелий.
- После выбора ингредиента сайт покажет, с чем он сочетается.
- Можно выбрать до трех ингредиентов.

## Подсказки

- Зеленые эффекты подходят для зелий.
- Красные эффекты подходят для ядов.
- Если ингредиент разделен на красную и зеленую части, нажмите на нужную сторону.
- Кнопка "Назад" возвращает к полной таблице после просмотра конкретного эффекта.
- "Удалить все" очищает выбранные ингредиенты.
