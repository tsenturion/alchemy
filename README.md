# Алхимия Skyrim

Русский справочник по алхимии Skyrim для подбора зелий, ядов и совместимых ингредиентов.

Сайт: <https://tsenturion.github.io/alchemy/>

## Что умеет сайт

- Показывает ингредиенты и их четыре алхимических эффекта.
- Ищет ингредиенты по названию.
- Ищет ингредиенты по одному или нескольким эффектам.
- Подсказывает, что получится при варке выбранных ингредиентов.
- Показывает подходящие сочетания и исключает варианты с противоположным результатом.
- Позволяет включать и отключать DLC, Creation Club, CACO и другие дополнения.

## Цвета

- Зеленый цвет относится к зельям.
- Красный цвет относится к ядам.
- Красно-зеленая ячейка означает, что у ингредиента есть и полезные, и вредные эффекты.
- Контрастная подсветка показывает важные совпадающие эффекты.

## Поиск и фильтр по эффектам

Нажмите `Выбрать Эффект`, `Эффект 1` или подсказку в основной таблице, чтобы открыть меню эффектов. После выбора таблица покажет только ингредиенты с этим эффектом.

После первого эффекта можно добавить второй, третий и четвертый эффект. Для этого используйте появляющиеся кнопки `+ Эффект 2`, `+ Эффект 3`, `+ Эффект 4` или подсказку под заголовками таблицы.

При добавлении второго и следующих эффектов меню показывает только те эффекты, для которых есть совпадения с уже выбранным набором. Кнопка `Назад` убирает последний выбранный эффект.

Нажатие на эффект внутри таблиц тоже открывает таблицу ингредиентов с этим эффектом.

## Выбор ингредиентов

Нажмите на название ингредиента, чтобы добавить его в выбранные. Можно выбрать до трех ингредиентов.

Если ячейка ингредиента разделена на красную и зеленую части, нажимайте на нужную сторону:

- красная сторона - для варки яда;
- зеленая сторона - для варки зелья.

В таблице `Вы выбрали` первая строка показывает, какие эффекты получатся при варке. Кнопка `Удалить` убирает один ингредиент, `Удалить все` очищает выбор полностью.

## Таблица сочетаний

После выбора ингредиента появляется таблица совместимых вариантов. В заголовке указано, с какими выбранными ингредиентами идет сравнение, сколько найдено вариантов и какие противоположные совпадения исключены.

Пример:

```text
Сочетается с Ингредиент 1 / Ингредиент 2, 12. Исключены положительные совпадения
```

Знак `/` означает логическое "или". Для ядов сайт исключает положительные совпадения, для зелий - отрицательные.

Ингредиенты с большим числом совпадающих эффектов идут выше. Совпадающие эффекты сдвигаются влево и подсвечиваются.

## Дополнения

В меню `Дополнения` можно включать и отключать наборы данных. Рядом с каждым пунктом показано количество ингредиентов.

Стандартный набор данных находится первым. При его отключении также отключаются скрытые дополнения, которые входят в стандартный набор по умолчанию.
