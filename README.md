# Алхимия Skyrim

Русский справочник по алхимии Skyrim для подбора зелий и ядов.

Сайт: <https://tsenturion.github.io/alchemy/>

## Что можно делать

- Смотреть ингредиенты и их четыре алхимических эффекта.
- Искать ингредиенты по названию.
- Искать эффекты через меню "Выбрать Эффект".
- Включать и отключать DLC, Creation Club, CACO и другие дополнения.
- Выбирать до трех ингредиентов и сразу видеть, что получится при варке.
- Смотреть, какие ингредиенты сочетаются с уже выбранными.

## Как читать таблицу

- Зеленый цвет относится к зельям.
- Красный цвет относится к ядам.
- Если ячейка разделена на красную и зеленую части, у ингредиента есть и полезные, и вредные эффекты.
- Верхняя строка под заголовками подсказывает, какая сторона используется для варки яда или зелья.
- Число рядом с поиском показывает, сколько ингредиентов сейчас отображается.

## Выбор ингредиентов

Нажмите на название ингредиента в таблице, чтобы добавить его в выбранные. Если ингредиент разделен на красную и зеленую части, нажимайте на нужную сторону:

- красная сторона - для яда;
- зеленая сторона - для зелья.

После выбора появляется таблица "Вы выбрали". В первой строке написано, какие совпадающие эффекты уже получаются при варке.

Кнопка "Удалить" убирает один ингредиент. Кнопка "Удалить все" очищает выбор полностью.

## Таблица сочетаний

После выбора ингредиента сайт показывает таблицу подходящих сочетаний.

Строка над таблицей выглядит примерно так:

```text
Сочетается с Ингредиент 1 / Ингредиент 2. Исключены положительные совпадения.
```

Знак `/` означает "или": ингредиент сочетается с одним из выбранных ингредиентов.

Пояснение про исключенные совпадения показывает, что сайт не предлагает варианты, которые случайно дадут противоположный результат:

- для ядов исключаются положительные совпадения;
- для зелий исключаются отрицательные совпадения.

## Поиск по эффекту

Откройте "Выбрать Эффект" и начните вводить название эффекта в поле поиска. После выбора эффекта главная таблица покажет только ингредиенты с этим эффектом.

Вернуться к полной таблице можно кнопкой "Назад".

## Дополнения

В меню "Дополнения" можно включать и отключать наборы данных. Рядом с каждым пунктом показано количество ингредиентов.

Стандартный набор данных находится первым. При его отключении также отключаются связанные скрытые дополнения, которые включены вместе с ним по умолчанию.
